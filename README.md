# Task Manager
Aplicación web simple para gestionar tareas diarias de forma rápida y organizada.

## Fuentes
Aplicación de subida de archivos con carga de trabajo

## Funcionalidades
- Crear tareas
- Listar tareas
- Eliminar tareas
- Guardar tareas en el navegador (localStorage)

## Operaciones
Asignar roles
Agregar condiciones
Administrar rutas
Agenciar trabajos

## Tecnologías

- CSS
- JavaScript

## Conversiones 
Asignaciones
Misiones
